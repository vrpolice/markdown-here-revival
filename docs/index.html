Focused on the writing flow

Everything useful.
Nothing between you and the message.

Markdown Here Revival stays inside Thunderbird's compose window and turns familiar Markdown into email-ready HTML.

01

Preview while you write

A resizable split view updates with your draft, so headings, lists, quotes, tables, and links never require guesswork.

02

Code that stays readable

Highlight more than 100 languages and choose from over 90 themes for technical email.

03

Math, tables, and emoji

Use LaTeX math, GitHub-Flavored Markdown tables, and emoji autocomplete without leaving the editor.

04

Made for sending, not just previewing

HTML is sanitized before delivery, source Markdown can survive in sent mail for later editing, and rendering happens automatically when you send.

How it works

From plain text to polished email in three steps.

  1. 1

    Compose

    Write with the Markdown syntax you already know.

  2. 2

    Preview

    Check the rendered result live in the adjacent pane.

  3. 3

    Send

    The extension safely replaces the draft with styled HTML.

Install manually

Ready in less than a minute.

Markdown Here Revival is distributed as an XPI while the project prepares for the Thunderbird Add-ons store.

Open the latest release →
  1. 01

    Download markdown-here-revival.xpi from Releases.

  2. 02

    Open Thunderbird's Add-ons Manager and choose Install Add-on From File…

  3. 03

    Open a compose window and use the M↓ toolbar button or Ctrl+Alt+M.
