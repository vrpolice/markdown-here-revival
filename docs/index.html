Everything You Need for Markdown Emails

Compose in plain text, send in rich HTML — with all the Markdown features you expect.

🔍

Live Preview Pane

See your rendered email as you type — a split-view preview updates on every keystroke. Resize it however you like.

🎨

Syntax Highlighting

100+ languages and 90+ themes via highlight.js. Perfect for sharing code snippets in emails.

📐

LaTeX Math Rendering

Inline and block math via TeXZilla (local, private) or CodeCogs (remote). Great for academic or technical discussions.

😄

Emoji Autocomplete

Type :smile: and get a popup picker. Full emoji shortcode support built right in.

📋

GFM Tables

GitHub-Flavored Markdown tables with extended syntax. Data looks great in any email client.

🔗

Smart Auto-Linking

Configurable bug/issue link patterns — #123 becomes a clickable link to your tracker.

✏️

Edit as New Message

Right-click any sent Markdown email and edit it as raw Markdown again. Embedded source survives sending.

🛡️

Security First

All HTML is sanitized through DOMPurify. Restrictive CSP in preview iframes. No eval() anywhere.

🌙

Dark Mode Ready

Theme-aware icons and CSS. Your rendered emails look great in light or dark Thunderbird.

How It Works

Three simple steps from Markdown to a beautifully formatted email.

1

Compose in Markdown

Write your email using Markdown syntax — headers, lists, code blocks, links, and more.

2

Preview in Real-Time

Watch the live preview pane update as you type. Toggle to classic mode if you prefer.

3

Send as Rich HTML

Hit send. The extension swaps in rendered HTML automatically. Your recipient sees a beautiful email.

Installation

Get up and running in under a minute.

1

Download .xpi from Releases

2

Open Thunderbird → Add-ons Manager → ⚙ → Install Add-on From File…

3

Open a compose window. Look for the M↓ button in the toolbar, or press Ctrl+Alt+M

4

Start writing in Markdown. The live preview pane appears automatically.

Requirements: Thunderbird 128 or later. Always check Releases for the latest compatible version.

Not on the Thunderbird Add-ons store yet — install manually via the .xpi file.

Project History

A decade of Markdown-to-email evolution, now updated for modern Thunderbird.

🥇

1st Gen (2012 – 2019)

Adam Pritchard created the original Markdown Here — a cross-browser extension for Chrome, Firefox, Safari, and Thunderbird. Maintenance ended in 2019 as browser APIs evolved.

🥈

2nd Gen (2021 – 2025)

Rob Lemley (JFX) forked and rewrote the extension specifically for Thunderbird using modern MailExtensions APIs. Introduced live preview, 13 languages, and the notification bar.

🥉

3rd Gen (2026 – today)

This fork keeps the extension compatible with new Thunderbird ESR releases, ships a refined professional business-style CSS, fixes UI regressions, and provides ongoing maintenance.
